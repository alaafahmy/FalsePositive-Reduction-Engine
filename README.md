# ZeroFalse — SAST False Positive Reduction

## Overview

ZeroFalse is a Visual Studio Code extension designed to reduce false positives in static application security testing (SAST) results generated by CodeQL.

The system enhances traditional static analysis by incorporating a Large Language Model (LLM) to perform semantic validation of detected vulnerabilities.

---

## Problem Statement

Static analysis tools such as CodeQL often produce a high number of false positive alerts due to their reliance on pattern-based detection rules.

These false positives:
- Increase developer workload
- Reduce trust in analysis tools
- Lead to alert fatigue

---

## Proposed Solution

ZeroFalse introduces a hybrid analysis approach:

1. CodeQL generates vulnerability alerts in SARIF format  
2. The system extracts contextual code information  
3. A Large Language Model (LLM) analyzes the context semantically  
4. Alerts are reclassified as:  
   - True Positive  
   - False Positive  

---

## System Architecture

CodeQL → SARIF → Context Extraction → LLM → Classification → VS Code UI

---

## Features

- Analyze SARIF files generated by CodeQL  
- Reduce false positive alerts using LLM reasoning  
- Visualize results directly inside VS Code  
- Provide contextual annotations for each alert  

---

## Current Limitations

- The extension currently supports Java only  
- Execution is limited to development mode (not yet packaged for marketplace)

## Requirements

### 1. Visual Studio Code
- Version 1.85 or higher  

---

### 2. Node.js
- Version 18 or higher  

Verify installation:  
node -v  
npm -v  

---

### 3. CodeQL CLI
- Installed and configured in system PATH  

Verify installation:  
codeql version  

---

### 4. Large Language Model (LLM)

Local Model (Recommended):  
- Install Ollama  
- Pull a model such as llama3  

Commands:  
ollama pull llama3  
ollama run llama3  

---

### 5. Project Dependencies

npm install  

---

### 6. Build the Extension

npm run compile  

---

## Usage

### Running the Extension (Development Mode)

1. Open the project in Visual Studio Code  
2. Open the file extension.ts  
3. Press F5 to launch the Extension Development Host  
4. A new VS Code window will open  

---

### Running Analysis

1. In the new window, open the project or folder that contains the dataset to be analyzed  
2. Ensure the dataset (e.g., SARIF file) is available  
3. Run the command:  
   ZeroFalse: Analyze SARIF File  
4. View refined results inside the extension interface  

---
